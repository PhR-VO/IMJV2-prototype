# IMJV2-prototype

## Opstarten

Start de lokale prototype-server vanuit deze map:

```bash
python3 server.py 8001
```

Open daarna:

```text
http://127.0.0.1:8001/
```

De exploitatielijst is rechtstreeks bereikbaar via:

```text
http://127.0.0.1:8001/?view=exploitations
```

Stop de server met `Ctrl+C` in het terminalvenster waarin hij draait.
